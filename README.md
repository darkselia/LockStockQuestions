# LockStockQuestions

## Описание
LockStockQuestions — неофициальный каталог вопросов из шоу «LockStock. Ставка на знания».
Приложение собирает вопросы, подсказки и ответы по выпускам, чтобы удобно просматривать их в одном месте и быстро переходить
к нужному эпизоду.

### Что делает приложение
- Показывает список всех выпусков и количество вопросов в каждом.
- Открывает страницу выпуска с полным набором вопросов.
- Для каждого вопроса отображает две подсказки, ответ и комментарий к ответу (если он есть).
- Позволяет раскрывать подсказки и ответы по шагам, не спойлеря сразу.
- Имеет быстрые переходы «Случайный выпуск» и «Случайный вопрос».

Проект создан в образовательных и некоммерческих целях и не связан с создателями шоу.

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone <URL вашего репозитория>
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd LockStockQuestions
   ```

3. Установите зависимости:
   ```bash
   npm install
   ```

## Скрипты

- **Запуск в режиме разработки:**
  ```bash
  npm run dev
  ```

- **Сборка проекта:**
  ```bash
  npm run build
  ```

- **Предпросмотр собранного проекта:**
  ```bash
  npm run preview
  ```

- **Проверка типов:**
  ```bash
  npm run type-check
  ```

- **Линтинг кода:**
  ```bash
  npm run lint
  ```
- **Преобразование таблицы вопросов из CSV в JSON:**
  ```bash
  py scripts/convert_questions.py data/questions.csv data/questions.json
  ```

## Используемые технологии

### Основные зависимости:
- **Vue 3**: Фреймворк д  я создания пользовательских интерфейсов.
- **Vue Router**: Для маршрутизации.
- **Pinia**: Состояние приложения.
- **Vuetify**: Компонентная библиотека для стилизации.
- **@mdi/font**: Иконки Material Design.

### DevDependencies:
- **Vite**: Быстрый инструмент для сборки.
- **TypeScript**: Статическая типизация.
- **ESLint**: Инструмент для анализа кода.
- **eslint-plugin-vue**: Плагин для линтинга Vue.
- **@vue/eslint-config-typescript**: Конфигурация ESLint для TypeScript.
- **@vue/eslint-config-prettier**: Интеграция Prettier с ESLint.
- **vue-tsc**: Проверка типов для Vue.
- **npm-run-all2**: Утилита для запуска нескольких npm-скриптов.

## Структура проекта

- **src/**: Исходный код приложения.
  - **assets/**: Стили и ресурсы.
  - **components/**: Компоненты Vue.
  - **router/**: Настройки маршрутизации.
  - **stores/**: Хранилище состояния (Pinia).
  - **views/**: Основные страницы приложения.

- **public/**: Публичные файлы, доступные без сборки.
